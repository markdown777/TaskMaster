# TaskMaster v3.5.0 升级说明 (Release Notes)

尊敬的用户，您好！

TaskMaster 从 v3.0.0 跃升至 v3.5.0，我们迎来了令人激动的 **AI 智能化时代**！本次更新不仅引入了强大的自然语言解析能力，还在底层重构了多端存储适配器，并采用了业界顶级的硬件加密标准来保护您的 API 密钥安全。

## 🚀 核心升级亮点

### 1. ✨ AI 智能自然语言解析
- **功能说明**：新增“✨ AI 模式”。只需在输入框中输入一段自然语言（如：“广东软件行业协会定于2026年4月28日（星期二）下午在广州召开2026年会员代表大会”），系统会在后台自动调用大语言模型，精准识别事件、时间、地点、优先级及核心要点，一键自动填充并创建项目。
- **信息追溯**：AI 创建任务时，会自动将您发送的原始自然语言记录到该任务的“备注”中（如 `[AI自动生成] 原始信息：...`），方便日后随时查看和比对原始上下文。
- **模型支持**：内置支持接入高性价比的 **DeepSeek**（默认推荐）与 **OpenAI** 接口，甚至支持自定义兼容格式的 Base URL。
- **贴心引导**：为新用户专门增加了呼吸灯动态光晕与引导 Tooltip，且附带了一键前往 DeepSeek 开放平台免费申请 API Key 的直达入口。

### 2. 🔐 BYOK 模式与银行级硬件加密
- **背景痛点**：传统的第三方插件常会明文存储用户的 API Key，极易遭受 XSS 跨站脚本窃取和本地木马扫盘。
- **加密方案 (AES-256-GCM)**：在 v3.5.0 中，我们引入了基于 Web Crypto API 的底层硬件加密机制。用户设置的 4 位安全 PIN 码将通过 PBKDF2 派生，使用 AES-256-GCM 算法对 API Key 进行深度加密后才落盘存储。
- **内存态调用 (Session Storage)**：解密后的 Key 仅存在于 Manifest V3 特有的 `chrome.storage.session` 运行内存中。浏览器关闭后，密钥立刻销毁。真正做到防窃取、防逆向。

### 3. 🎨 极简美学与交互升级
- **无缝融合**：重绘了全新的 PIN 码解锁弹窗。采用了高透的明亮毛玻璃遮罩 (`backdrop-filter`)、直角几何切割、大留白以及扩散性的轻盈弥散阴影，与 v3.0.0 的极简排版美学完美统一。
- **等宽数字排版**：针对 PIN 码输入，特别使用了严谨的等宽字体族 (Monospace) 并拉大了字符间距，赋予用户精密极客的输入质感。
- **阻尼微动效**：所有新按钮及弹窗的出现/消失，均遵循高级缓动曲线 (`cubic-bezier(0.16, 1, 0.3, 1)`) 和物理级的防跳动阻尼设计。
- **搜索体验重构**：将搜索框升级为优雅的悬浮弹出层（Popover），默认隐藏不占空间。点击搜索图标后自动聚焦，点击空白处自动关闭并清空搜索状态，最大化释放任务列表垂直空间。
- **任务列表排版优化**：重构了任务分组（如“置顶”、“活动”）的间距逻辑，去除了单条任务底部多余的分割线，使列表视觉更加紧凑连贯；底部操作栏改为吸底固定，布局更稳固。
- **输入框智能提示**：优化了输入框的占位符（Placeholder）颜色与字号。在普通模式下颜色加深提升辨识度；在 AI 模式下自动缩小字号并添加过渡动画，确保长提示语能完整显示。
- **归档列表卡片化**：将“已完成任务”彻底重构为高级的两列瀑布流卡片网格布局（Grid）。去除生硬边框，通过背景色块区分空间（去AI味）。默认仅展示最新归档的 6 条任务（完美的 3x2 矩阵），其余历史任务自动折叠，保持界面极度清爽。
- **分享功能现代化升级**：彻底废弃旧版内联样式和原生弹窗，将分享菜单重写为符合 v3.0.0 规范的毛玻璃模态框。修复了潜在的事件内存泄漏 Bug，并将分享内容引擎升级为优雅的 Markdown 排版（支持 Emoji、代码块和引用语法），分享到 Notion、飞书等工具时效果绝佳。

### 4. 🛠️ 底层架构演进 (为小程序铺路)
- **解耦设计**：为了未来能向微信小程序等多设备平台迁移，v3.5.0 对底层代码进行了重度抽象。
- **新增适配器**：全面引入了 `StorageAdapter` (存储隔离层)、`NetworkAdapter` (网络隔离层) 与 `CryptoAdapter` (加密隔离层)。这使得核心的 AI 服务 (`AIService`) 可以一行代码不改地直接复用到未来的多端项目中。
- **鲁棒性修复**：修复了 LLM 返回的非标时间格式（带斜杠、秒数或空格）可能引起的时区偏差问题，增强了输入时间清洗功能。
- **上下文兼容性修复**：全面修复了在某些扩展执行环境下（特别是 `options.js` 和后台服务中）`chrome.runtime.lastError` 属性丢失导致的“未定义”异常，增强了数据存取和 API 密钥加密存储时的容错与健壮性。
