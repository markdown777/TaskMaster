Edge Review Team,

TaskMaster v3.0.0 is a productivity tool for task management with cross-device sync and local encryption.

### 1. Test Accounts & Dependencies
- Accounts: Not required. Fully operational offline. No registration needed.
- Dependencies: None. Relies entirely on native browser APIs.

### 2. How to Test
1. Pin extension to toolbar and click icon.
2. Type a task (e.g. "Test Task") and click "Time" to set due date.
3. Click "Create".
4. Click "Settings" (bottom) to open Options page.
5. Test the "Export Data" button to verify JSON backup feature.

### 3. Permissions Justification
- `storage`: Required to save user tasks and settings. Uses `chrome.storage.local` for full data and `chrome.storage.sync` for lightweight cross-device syncing.
- `alarms`: Required to run background timers that trigger daily automated backups and task deadline notifications, even when popup is closed.
- `notifications`: Required to alert users via system desktop notifications when tasks are due.
- `downloads`: Required ONLY on the Options page when the user clicks "Export Data". It allows the extension to save the user's task data as a local JSON backup file.

### 4. Privacy & Security
- All data is stored locally or synced via the native Edge Sync API.
- NO analytics, tracking scripts, or telemetry included.
