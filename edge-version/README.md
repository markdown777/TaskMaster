# TaskMaster - 智能待办事项管理扩展

TaskMaster 是一个功能强大的 Chrome 扩展程序，专注于帮助用户高效管理待办事项。通过直观的界面和智能的任务管理功能，让您的工作和生活更有条理。

## 功能特性

- 📝 智能任务管理：轻松创建、编辑和组织待办事项
- ⏰ 任务提醒：支持设置任务提醒，不错过重要事项
- 🔔 桌面通知：及时接收任务提醒通知
- ⚙️ 自定义设置：根据个人需求定制使用体验
- 💾 数据同步：使用 Chrome 存储 API 确保数据安全保存
- 🎨 简洁界面：现代化的用户界面设计，操作简单直观

## 安装说明

1. 从 Chrome 网上应用店安装：
   - 访问 Chrome 网上应用店
   - 搜索 "TaskMaster"
   - 点击 "添加到 Chrome" 即可完成安装

2. 开发者安装：
   ```bash
   # 克隆仓库
   git clone https://cnb.cool/jxuz/ChromePlus.git
   
   # 安装依赖
   npm install
   
   # 在 Chrome 浏览器中加载扩展：
   # 1. 打开 Chrome 浏览器
   # 2. 访问 chrome://extensions/
   # 3. 开启"开发者模式"
   # 4. 点击"加载已解压的扩展程序"
   # 5. 选择项目目录
   ```

## 使用指南

1. 点击浏览器工具栏中的 TaskMaster 图标打开扩展
2. 点击 "+" 按钮添加新的待办事项
3. 为任务设置提醒时间（可选）
4. 通过选项页面（右键扩展图标 -> 选项）进行个性化设置

## 技术栈

- Chrome Extension Manifest V3
- JavaScript (ES6+)
- Chrome Storage API
- Chrome Notifications API
- Chrome Alarms API
