# TaskMaster - 智能任务管理器 (Microsoft Edge版本)

## 概述
TaskMaster是一款功能强大的智能待办事项管理扩展，专为Microsoft Edge浏览器优化。帮助您高效管理日常任务，提升工作效率。

## 主要功能

### 📝 任务管理
- 快速创建、编辑和删除任务
- 设置任务截止日期和优先级
- 标记任务完成状态
- 任务归档功能

### 🔍 智能搜索
- 实时搜索任务内容
- 按状态筛选任务（活跃/已完成）
- 仅显示带笔记的任务
- 高亮显示搜索关键词

### 📋 笔记功能
- 为每个任务添加详细笔记
- 支持富文本编辑
- 笔记内容搜索

### ⏰ 提醒系统
- 智能任务提醒
- 自定义提醒时间
- 浏览器通知支持

### 📤 数据管理
- 任务数据导入/导出
- 云端同步存储
- 数据备份功能

### 🎨 用户体验
- 现代化界面设计
- 响应式布局
- 深色/浅色主题切换
- 快捷键支持

## 系统要求
- Microsoft Edge 88或更高版本
- Windows 10/11或macOS

## 安装方法

### 从Microsoft Edge Add-ons商店安装（推荐）
1. 打开Microsoft Edge浏览器
2. 访问Edge Add-ons商店
3. 搜索"TaskMaster"
4. 点击"获取"按钮安装

### 手动安装开发版本
1. 下载扩展文件包
2. 打开Edge浏览器，进入`edge://extensions/`
3. 开启"开发人员模式"
4. 点击"加载解压缩的扩展"
5. 选择扩展文件夹

## 使用指南

### 创建任务
1. 点击浏览器工具栏中的TaskMaster图标
2. 在弹出窗口中输入任务标题
3. 设置截止日期（可选）
4. 点击"添加任务"按钮

### 管理任务
- **完成任务**：点击任务前的复选框
- **编辑任务**：点击任务标题进行编辑
- **添加笔记**：点击"笔记"按钮
- **删除任务**：点击删除图标

### 搜索任务
1. 在搜索框中输入关键词
2. 使用筛选选项精确查找
3. 查看高亮显示的搜索结果

### 设置提醒
1. 打开扩展设置页面
2. 配置提醒时间和频率
3. 确保浏览器通知权限已开启

## 数据隐私
- 所有数据存储在本地浏览器中
- 支持Chrome同步账户数据同步
- 不收集用户个人信息
- 完全离线工作
