## 🚀 提交步骤

### 步骤1: 打包扩展
```powershell
# 在edge-version目录下运行
.\package-edge.ps1
```
这将生成 `TaskMaster-Edge-v2.9.0.zip` 文件。

### 步骤2: 注册开发者账户
1. 访问 [Microsoft Partner Center](https://partner.microsoft.com/dashboard/microsoftedge/overview)
2. 使用Microsoft账户登录
3. 完成开发者注册流程
4. 支付一次性注册费用 (约$9 USD)

### 步骤3: 创建新扩展提交
1. 在Partner Center中点击"Create new extension"
2. 上传ZIP文件 (`TaskMaster-Edge-v2.9.0.zip`)
3. 填写扩展信息

### 步骤4: 填写扩展详情

#### 基本信息
- **扩展名称**: TaskMaster - 智能任务管理器
- **简短描述**: 功能强大的智能待办事项管理扩展
- **详细描述**: 
```
功能强大的智能待办事项管理扩展，支持任务创建、提醒、搜索、笔记和数据导入导出等功能。

主要特性：
✅ 智能任务管理 - 快速创建、编辑和组织任务
🔍 强大搜索功能 - 实时搜索和智能筛选
📝 笔记系统 - 为任务添加详细笔记
⏰ 提醒功能 - 自定义任务提醒
📤 数据同步 - 云端存储和备份
🎨 现代界面 - 简洁美观的用户体验

适用于学生、职场人士和任何需要高效管理待办事项的用户。
```

#### 分类和标签
- **主要分类**: Productivity
- **标签**: 任务管理, 待办事项, 提醒, 笔记, 效率工具

#### 年龄分级
- **年龄等级**: General audiences
- **内容描述**: 无不当内容，适合所有年龄用户

### 步骤5: 上传截图和图标

#### 必需的图像资源
1. **扩展图标** (128x128 PNG)
   - 使用 `icons/icon128.png`
   
2. **商店截图** (至少1张，最多10张)
   - 推荐尺寸: 1280x800 或 1920x1080
   - 展示主要功能界面
   - 建议截图内容:
     - 主界面任务列表
     - 任务创建界面
     - 搜索功能演示
     - 笔记编辑界面
     - 设置页面

3. **宣传图片** (可选)
   - 尺寸: 1400x560
   - 用于商店推广展示

### 步骤6: 隐私和合规

#### 隐私政策
```
隐私政策要点：
- 所有数据存储在用户本地浏览器中
- 不收集用户个人信息
- 不向第三方传输数据
- 支持Chrome同步功能（可选）
- 完全离线工作
```

#### 权限说明
- **storage**: 用于保存任务数据
- **alarms**: 用于任务提醒功能
- **notifications**: 用于显示提醒通知

### 步骤7: 测试和验证
1. 上传后等待自动验证
2. 修复任何发现的问题
3. 确保所有功能正常工作

### 步骤8: 提交审核
1. 检查所有信息无误
2. 点击"Submit for review"
3. 等待Microsoft审核（通常1-7个工作日）

## 📝 审核要点

### Microsoft Edge Add-ons政策要求
1. **功能性**: 扩展必须按描述正常工作
2. **安全性**: 无恶意代码或安全漏洞
3. **用户体验**: 界面友好，功能直观
4. **内容政策**: 符合Microsoft内容政策
5. **技术标准**: 遵循Web标准和最佳实践

### 常见拒绝原因及解决方案
1. **权限过度**: 已最小化权限请求
2. **功能不明确**: 已提供详细功能说明
3. **图标质量**: 使用高质量PNG图标
4. **描述不准确**: 已提供准确的功能描述
